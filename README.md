niftycloud
=========

Ansible role for importing module of NIFTYCloud

Requirements
------------

* Distribution = CentOS / Ubuntu
* python >= 2.6 (localhost)

Role Variables
--------------

None.

Dependencies
------------

None.

Example Playbook
----------------

* Load libraries
```
hosts: servers
roles:
  - niftycloud
```

* How to use libraries?
  * Refer to [README.md](library/README.md) of ansible modules for NIFTY Cloud
